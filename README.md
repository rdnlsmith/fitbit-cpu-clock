# CPU

Digital clock face for Fitbit Versa, inspired by retro computer ads. Currently in early development.

![Screenshot](./screenshot_github.png)

## Current Features

* Time (12 / 24 hr)
* Day and date
* Heart rate
* Step count

## Planned Features

* Weather
* Other activity stats (configurable)
* Heart rate animation

# Acknowledgements
